## 🧠 Overview

**MianSnap** is a fast, browser-based thumbnail creation tool that transforms video frames into high-converting thumbnails in seconds.

Built under the **MultiMian** ecosystem, it runs entirely in the browser — meaning:

* ⚡ No uploads
* 🔒 100% privacy
* 🚀 Instant performance

It combines video frame capture, canvas editing, and smart automation to help creators design thumbnails faster than ever.

---

## ✨ Features

### 🎥 Video Frame Capture

* Upload and preview videos
* Frame-by-frame navigation
* Capture exact moments with precision

---

### 🎨 Canvas Editor (Fabric.js)

* Add text, images, and shapes
* Drag, resize, rotate elements
* Apply shadows, strokes, and effects

---

### 🧱 Layer Panel

* Manage all canvas elements visually
* Reorder layers (z-index)
* Toggle visibility 👁️
* Lock/unlock objects 🔒

---

### 🔤 Smart Typography

* Auto glow & shadow for better readability
* Dynamic contrast adjustment
* Clean, bold text styling

---

### 🟥 Safe-Zone Overlays

* Avoid placing content in blocked areas
* Supports:

  * YouTube
  * TikTok
  * Instagram

---

### 💾 Project Saving (IndexedDB)

* Auto-save projects locally
* Load previous work anytime
* Store branding kits (logos, colors, fonts)

---

### 📦 Asset Manager

* Upload and reuse assets
* Save brand identity
* One-click reuse system

---

### 📤 Export System

* Export thumbnails in:

  * 720p (fast)
  * 1080p (high quality)
* Formats:

  * JPG
  * PNG
